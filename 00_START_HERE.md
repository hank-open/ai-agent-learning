#### Step 2: 第一个 LLM 调用 (15 min)

创建 `hello_agent.py`：

```python
from anthropic import Anthropic

client = Anthropic()

response = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=100,
    messages=[{
        "role": "user",
        "content": "你好，请自我介绍一下"
    }]
)

print(response.content[0].text)
```

运行：
```bash
python hello_agent.py
```

## 🎓 关键概念速览

如果你没有时间读文档，至少要理解这些：

### 什么是 Agent?

```
Agent = 一个能够思考、使用工具、学习的 AI 系统

它的工作流程:
1. 接收用户问题
2. 思考需要什么信息/工具
3. 调用适当的工具
4. 观察工具的结果
5. 根据结果调整策略
6. 重复 2-5 直到问题解决
7. 给出最终答案
```

### 为什么 Agent 很重要?

```
传统 ChatBot:
用户 → 固定规则 → 回复
(局限性: 只能回答预定义的问题)

AI Agent:
用户 → 思考 → 使用工具 → 思考 → 使用工具 → ... → 回复
(优势: 能解决动态、复杂的问题)

例子:
Q: "帮我分析上周的销售数据并生成报告"
Agent: 查询数据库 → 计算统计 → 生成图表 → 写总结 → 完成
```

### 你需要学什么?

```
基础层 (Week 1-2):
├─ LLM API 使用
├─ Prompt 工程
└─ Agent 核心概念 (ReAct 模式)

框架层 (Week 3-4):
├─ Python + FastAPI
├─ 数据库 (SQL + Vector)
└─ LangChain 框架

工程层 (Week 5-8):
├─ 可靠性设计 (错误处理、重试)
├─ 性能优化 (缓存、成本控制)
└─ 可观测性 (日志、监控)

实战层 (Week 9-20):
└─ 3-4 个完整项目
```

## ❓ 快速 FAQ

### Q: Python 不会怎么办?

**A:** 没关系！
- LLM 代码都比较直白
- 你已经懂异步编程
- 跟着 week1_detailed_plan.md 的代码学，会很快上手
- 3-4 周就能写出生产级代码

### Q: 需要多少前期投入？

**A:** 零。
- 所有工具都免费 (Python, FastAPI, 向量库)
- API 成本很低 (实验在 $10-50 范围)
- 有免费的 API 额度 (Anthropic 提供)

### Q: 最难的部分是什么？

**A:** 三个挑战：
1. **概念多** - Agent 有很多新概念，但 week1_detailed_plan.md 会逐步讲解
2. **需要持续投入** - 不是一周速成，但坚持会有结果
3. **从前端到全栈** - 需要学后端，但对你来说容易很多

### Q: 能找到工作吗？

**A:** 很可能！
- AI 工程师需求量大，薪资高
- 有 3+ 完整项目的作品集很有竞争力
- 需要: 项目 + 文档 + GitHub + 面试准备

### Q: 应该全职还是业余？

**A:** 看你的情况：
- **业余优势** (继续工作):
  - 保持现金流
  - 当前工作中可以学习前端 + LLM 的结合
  - 压力小，容易坚持
  - 需要 20-24 周

- **全职优势** (脱产学习):
  - 进度快，16 周完成
  - 专注力强，深度理解好
  - 能参与开源和社区
  - 经济压力大

### Q: 应该学 Python 还是 Node.js？

**A:** **必须是 Python**
- 95% LLM 社区都在 Python
- 库最丰富 (LangChain, LangGraph 等)
- 初创用 Python 较多
- 后续可学 Node.js 作备选

### Q: 跟其他前端工程师相比，我的优势是什么？

**A:** 巨大优势：
1. ✅ 已有 API 实战经验 (其他人从零开始)
2. ✅ 异步编程熟悉 (立即可用)
3. ✅ 全栈意图明确 (不像有人没方向)
4. ✅ 组件化思维 (Agent 设计会很顺)
5. ✅ UX 理解深 (其他人没有)
